
目标函数已打在  meta_test/quadsim/func_experiment.py

其中calculate_ACE()为目标函数，返回值即为评价指标ACE，其值越小越好

两个参数分别为update_pid, need_pdf

update_pid为包含所有PID参数的字典，在论文代码中给出了默认值
update_pid_template = {
        "MC_ROLLRATE_P": 0.4,
        "MC_PITCHRATE_P": 0.4,
        "MC_YAWRATE_P": 0.1,
        "MC_ROLLRATE_I": 0.07,
        "MC_PITCHRATE_I": 0.07,
        "MC_YAWRATE_I": 0.0005,
        "MC_ROLLRATE_D": 0.0016,
        "MC_PITCHRATE_D": 0.0016,
        "MC_YAWRATE_D": 0.01
    }
update_pid == None,  采用给出的默认值

need_pdf == true， 会将目标任务中的测试结果可视化，生成pdf，保存路径为quadsim/plots/执行日期/final



envs：
pytorch >=1.8



需要额外下载的包：
rowan 
1）pip：
	pip install rowan --user
2）conda：
	conda config --add channels conda-forge
	conda install rowan
